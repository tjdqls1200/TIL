# 수박수박수박수박수박수?

## Programmers, Level 1

길이가 n이고, 수박수박수박수....와 같은 패턴을 유지하는 문자열을 리턴하는 함수, solution을 완성하세요. 예를들어 n이 4이면 수박수박을 리턴하고 3이라면 수박수를 리턴하면 됩니다.

##### 제한 조건

- n은 길이 10,000이하인 자연수입니다.

  

## 1. 풀이 코드

```
class Solution {
    public String solution(int n) {
        String answer = "";

        if (n % 2 == 1) {answer += "수";}
        while (n > 1) {
            if (n % 2 == 0) {answer += "수박";}
            else {answer += "박수";}
            n -= 2;
        }
        return answer;
    }
}
```
