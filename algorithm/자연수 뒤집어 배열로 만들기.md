# 자연수 뒤집어 배열로 만들기

## Programmers ,Level 1

자연수 n을 뒤집어 각 자리 숫자를 원소로 가지는 배열 형태로 리턴해주세요. 예를들어 n이 12345이면 [5,4,3,2,1]을 리턴합니다.

## 1. 풀이 코드

```
import java.util.*;

class Solution {
    public int[] solution(long n) {
        int[] answer = new int[(int)(Math.log10(n) + 1)];
        int i = 0;
        while (n > 0) {
            answer[i++] = (int)(n % 10);
            n /= 10;
        }
        return answer;
    }
}
```
